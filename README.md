# How to Contribute

1. Clone this repository:

```sh
git clone https://github.com/react-spring/react-spring
cd react-spring
```

1. Bootstrap the packages:

```sh
npx lerna bootstrap
```

1. Link the packages:

```
npx lerna exec -- npm link
```

1. Link `react-spring` to your project:

```sh
cd ~/my-project
npm link react-spring
```

1. Let's get cooking!
